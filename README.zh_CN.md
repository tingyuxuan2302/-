<h1 align="center">Nest Admin</h1>

- 预览: https://qkstart.com

## 📦 下载

```bash
$ git clone https://gitee.com/weiqinke/nest-admin.git
$ cd nest-admin
$ npm install
$ npm start
```

## 🔨 构建

```bash
yarn install
yarn run build
```
